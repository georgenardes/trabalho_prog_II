Projeto da disciplina de programação ll

Sistema para cadastro, consulta e deleção de filmes.

Foi utilizado a linguagem c, no ambiente de desenvolvimento Code Blocks.

A navegação no sistema deve ser realizada utilizando o teclado (setas e ENTER)

Os filmes cadastrados são armazenados no arquivo dados.txt, utilizando uma linha para cada filme, separando os atributos pelo delimitador ";"
Esse arquivo deve estar junto do executável do sistema, ou junto ao arquivo do projeto, caso o programa seja executado pela IDE.

Os usuários com permissão para cadastro e deleção devem estar préviamente inseridos no admin.txt, utilizando uma linha para cada usuário, no estilo "<usuário;senha>". Ex.: admin;123.
